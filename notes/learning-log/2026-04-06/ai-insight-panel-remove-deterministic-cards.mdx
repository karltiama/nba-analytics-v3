## Context
The lower section in `AIInsightPanel` showed deterministic stat cards (pace/defense/trend snippets) that felt repetitive and less useful than the AI slate narrative.

## Goal
Remove the deterministic cards section so the sidebar focuses only on the AI slate summary.

## Files Changed
- `components/betting/AIInsightPanel.tsx`

## Implementation Notes (key logic explained)
- Removed the entire deterministic insights block from the panel body.
- Kept the existing summary block and loading/hint states unchanged.
- Updated footer copy to remove mention of stat cards.
- Added `void insights;` because the prop is still passed from parent, but currently not rendered.

## Key Decisions (and alternatives considered)
- Chose direct removal of the section rather than hiding with CSS to reduce UI complexity.
- Alternative considered: keep section collapsed by default; rejected to avoid adding another interaction control.

## Assumptions
- Sidebar should now act as a focused AI context panel, not a mixed AI + deterministic card stack.

## Failure Modes
- If users relied on deterministic cards, that information is no longer visible in this panel.
- Future lint settings may flag currently-unused helper logic if strict rules are enabled.

## How to Test (commands + steps)
1. Run app and open `/betting`.
2. Confirm `AI Insights` panel shows only slate summary + footer.
3. Confirm no “Highest Pace Team / Elite Defense Alert” cards render.
4. Confirm loading and empty-hint states still work.

## Next Steps
- If desired, remove now-unused insight card helpers/types in a follow-up cleanup diff.
