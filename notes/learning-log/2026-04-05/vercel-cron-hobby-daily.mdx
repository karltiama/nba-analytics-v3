## Context

Vercel Hobby rejects cron expressions that run more than once per day (e.g. `*/15 * * * *`).

## Goal

- **Goal:** Align `vercel.json` with Hobby limits so deploy succeeds.
- **Inputs:** Vercel plan constraints.
- **Output/Done:** Daily schedule; docs explain Hobby vs Pro.

## Files Changed

- `vercel.json` — `0 12 * * *` (12:00 UTC daily) for `/api/cron/paper-settle`.
- `docs/deployment-checklist.md` — Hobby vs Pro, how to use 15-minute cadence on Pro.

## Implementation Notes (key logic explained)

- **Trade-off:** Paper settlement runs once per day on Hobby instead of every 15 minutes. For faster settlement on Hobby, trigger `GET /api/cron/paper-settle` manually or from an external cron (GitHub Actions, etc.) with `Authorization: Bearer <secret>`.
